# 🎵 Spotify音乐数据分析项目

## 项目概述
基于114,000首Spotify歌曲的深度数据分析，探索音乐流行度的关键因素和流派特征模式。

## 🎯 项目亮点
- **数据规模**: 114,000首歌曲 × 114种流派
- **分析方法**: 统计分析 + 机器学习聚类
- **技术栈**: Python, Pandas, Matplotlib, Scikit-learn
- **商业价值**: 为音乐推荐和内容策略提供数据洞察

## 📊 关键发现
1. **高流行度流派**: Pop-Film, K-Pop, Chill音乐表现最佳
2. **流派聚类**: 114种流派自然分为4个特征群组  
3. **特征相关性**: 量化音频特征与流行度的关系

## 📁 文件说明
- `spotify_analysis.ipynb` - 完整的数据分析代码
- `*.png` - 生成的数据可视化图表
